🚀 Quick Start
[!WARNING]
These are my personal files. Blindly cloning might break your system. Check the scripts before execution.

1. Clone the repository
Bash
git clone [https://github.com/dead-Spy/dotfiles.git](https://github.com/dead-Spy/dotfiles.git)
cd dotfiles
2. Manual Installation
Copy the desired configuration to your .config directory:

Bash
cp -r .config/waybar ~/.config/
cp -r .config/hypr ~/.config/
3. Key Dependencies
Ensure you have these installed via pacman or yay:

waybar

hyprland

playerctl (for media scripts)

python-requests (for weather)

otf-font-awesome / ttf-jetbrains-mono-nerd

🛠️ Key Bindings (Brief)
Mod + Q — Kill Window

Mod + Space — App Launcher

Mod + Enter — Terminal

Mod + M — Exit Menu
