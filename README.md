## 🛰️ System Overview
A performance-driven, aesthetically curated **Arch Linux** environment focused on a seamless workflow using **Hyprland** and **Waybar**. Designed for modern full-stack development and minimalist cinematic UI.

### 🧩 Components
| Component | Choice |
| :--- | :--- |
| **Window Manager** | Hyprland (Wayland) |
| **Status Bar** | Waybar (Custom CSS) |
| **Terminal** | Foot / Alacritty |
| **Shell** | Fish / Starship Prompt |
| **Theme** | Gruvbox / Custom Dark |
| **Editor** | Neovim / VS Code |
